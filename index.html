Como surgiu a internet

A internet foi criada em 1969, nos Estados Unidos. Chamada de Arpanet, tinha como função

interligar laboratórios de pesquisa. Naquele ano, um professor da Universidade da Califórnia

passou para um amigo em Stanford o primeiro e-mail da história. Essa rede pertencia ao Departamento

de Defesa norte-americano.

Como funciona o HTML

O HTML é uma linguagem de marcação. Estas linguagens são constituídas de códigos que delimitam

conteúdos específicos, segundo uma sintaxe própria. O HTML tem códigos para criar paginas na web.

códigos que definem o tipo de letra, qual o tamanho, cor, espaçamento, e vários outros aspectos do site.

Tags e seus atributos

As tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve

um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o

conteúdo visual da página.Atributos HTML são palavras especiais usadas dentro da tag de abertura

para controlar o comportamento do elemento.
